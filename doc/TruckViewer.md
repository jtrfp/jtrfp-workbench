# Truck viewer

The truck viewer can be started from the *POD Explorer* view in the workbench.

## Screenshot

![Screenshot](images/screenshot06.png)

## Keys

* *Esc* exit
* *T* toggle face modes
  * textured
  * wireframe
* *Cursor up/down* rotate around X axis
* *Cursor left/right* rotate around Y axis
* *A/D* steer
